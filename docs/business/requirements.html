Бизнес-требования

Целевая аудитория​

Организаторы турниров​

КритерийОписание
Кто этоПредставители компьютерных клубов, администраторы Discord-сообществ, студенческие организаторы, любительские лиги
Основная цельПровести турнир с максимальной заполняемостью и минимальными организационными сложностями
Основные задачиОрганизация, продвижение, регистрация участников, формирование сетки, фиксация результатов
Основные болиНедостаточная регистрация, отмена турниров, ручное ведение сетки, сложность продвижения
Текущие инструментыDiscord, Google-таблицы, соцсети

Игроки​

КритерийОписание
Кто этоШкольники, студенты, игроки любительского уровня, начинающие киберспортсмены
Основная цельНайти турнир своего уровня и проявить себя
Основные задачиПоиск турнира, регистрация, участие, отслеживание результатов
Основные болиНет единого места поиска, сложно найти LAN-турнир, нет фильтрации по уровню, отсутствие статистики
Текущие инструментыDiscord-каналы, соцсети, рекомендации друзей

Анализ конкурентов​

КонкурентКлючевые особенностиНедостатки
FACEITMatchmaking платформа, турниры по разным дисциплинам, рейтинг ELOНельзя создать LAN-турнир, только Online, хорошо развит только CS-сегмент
ChallongeПростой веб-сервис для генерации турнирных сетокПлохой поиск турниров для игроков, ограниченный функционал, неудобный UX
VSCL.ruРоссийская платформа, есть LAN и Online турнирыНельзя создать свой турнир, ограничение по региону, только CS/Dota2

Ключевое отличие T-Esports — возможность самостоятельного создания LAN и Online турниров с одновременной публикацией в централизованном каталоге.
