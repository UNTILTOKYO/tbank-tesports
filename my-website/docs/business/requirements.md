# Бизнес-требования

## Целевая аудитория

### Организаторы турниров

| Критерий | Описание |
|---|---|
| Кто это | Представители компьютерных клубов, администраторы Discord-сообществ, студенческие организаторы, любительские лиги |
| Основная цель | Провести турнир с максимальной заполняемостью и минимальными организационными сложностями |
| Основные задачи | Организация, продвижение, регистрация участников, формирование сетки, фиксация результатов |
| Основные боли | Недостаточная регистрация, отмена турниров, ручное ведение сетки, сложность продвижения |
| Текущие инструменты | Discord, Google-таблицы, соцсети |

### Игроки

| Критерий | Описание |
|---|---|
| Кто это | Школьники, студенты, игроки любительского уровня, начинающие киберспортсмены |
| Основная цель | Найти турнир своего уровня и проявить себя |
| Основные задачи | Поиск турнира, регистрация, участие, отслеживание результатов |
| Основные боли | Нет единого места поиска, сложно найти LAN-турнир, нет фильтрации по уровню, отсутствие статистики |
| Текущие инструменты | Discord-каналы, соцсети, рекомендации друзей |

## Анализ конкурентов

| Конкурент | Ключевые особенности | Недостатки |
|---|---|---|
| FACEIT | Matchmaking платформа, турниры по разным дисциплинам, рейтинг ELO | Нельзя создать LAN-турнир, только Online, хорошо развит только CS-сегмент |
| Challonge | Простой веб-сервис для генерации турнирных сеток | Плохой поиск турниров для игроков, ограниченный функционал, неудобный UX |
| VSCL.ru | Российская платформа, есть LAN и Online турниры | Нельзя создать свой турнир, ограничение по региону, только CS/Dota2 |

**Ключевое отличие T-Esports** — возможность самостоятельного создания LAN и Online турниров с одновременной публикацией в централизованном каталоге.

## Ключевые стейкхолдеры

### Внутренняя команда

| Стейкхолдер | Роль |
|---|---|
| Заказчик / Владелец продукта | Определяет цели, бюджет, стратегию развития |
| Продакт-менеджер | Формирует требования и приоритеты |
| Backend / Frontend / Tech Lead | Реализация технической части |
| UX/UI-дизайнер | Проектирование пользовательского интерфейса |
| QA | Тестирование платформы перед релизом |
| Маркетинг | Привлечение пользователей |

### Внешние стейкхолдеры

| Стейкхолдер | Влияние |
|---|---|
| Организаторы турниров | Формируют спрос на функционал, влияют на репутацию и рост платформы |
| Игроки и команды | Пользуются ключевой фичей, финансируют платформу через fee за участие |
