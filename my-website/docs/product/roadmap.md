# Версии и план развития

## Приоритизация по MoSCoW

### Управление пользователями

| Приоритет | Функционал |
|---|---|
| Must Have (MVP) | Регистрация, авторизация, роли |
| Should Have (2.0) | Профиль с минимальным набором кастомизаций |
| Could Have (3.0) | Расширенные настройки аккаунта |
| Won't Have (4.0+) | Авторизация через соцсети |

### Управление турнирами

| Приоритет | Функционал |
|---|---|
| Must Have (MVP) | Создание и публикация турнира |
| Should Have (2.0) | Редактирование турнира, продвижение турниров |
| Could Have (3.0) | Платёжная система (депозиты и призовые) |
| Won't Have (4.0+) | Персональные рекомендации |

### Турнирная логика

| Приоритет | Функционал |
|---|---|
| Must Have (MVP) | Генерация сетки Single Elimination, ввод результатов |
| Should Have (2.0) | Поддержка Best of 3 |
| Could Have (3.0) | Double Elimination, Team Deathmatch, рейтинг ELO |
| Won't Have (4.0+) | Продвинутая аналитика профиля |

### Аналитика

| Приоритет | Функционал |
|---|---|
| Must Have (MVP) | История участия |
| Should Have (2.0) | Базовая статистика |
| Could Have (3.0) | Рейтинг ELO |
| Won't Have (4.0+) | KDA, матчи, винрейт, любимая карта |

## Сроки реализации

### MVP (6 месяцев)

| Месяц | Задачи |
|---|---|
| Месяц 1 | Анализ рынка и проектирование архитектуры |
| Месяц 2–3 | Разработка эпиков: регистрация, создание турнира, каталог |
| Месяц 4 | Реализация турнирной логики (сетка, результаты) |
| Месяц 5 | Тестирование и исправление ошибок |
| Месяц 6 | Запуск MVP, сбор обратной связи, первичное привлечение пользователей |

### Версия 2.0 (3–4 месяца после MVP)
Профиль с кастомизациями, редактирование турниров, фильтр по уровню, Best of 3, базовая статистика, интеграция с Discord.

**Цель:** повышение удержания пользователей и конкурентоспособности платформы.

### Версия 3.0 (4–6 месяцев после 2.0)
Платёжная система, расширенный поиск, лист ожидания, Double Elimination, рейтинг ELO.

**Цель:** запуск устойчивой бизнес-модели и достижение окупаемости.

### Версия 4.0+ (6+ месяцев после 3.0)
Авторизация через соцсети, интеграция с игровыми трекерами (SteamID, Faceit, Dotabuff), продвинутая аналитика, мобильное приложение, географическое масштабирование.
